# Tale of Nowhere soundboard
A lightweight soundboard for the wonderful Tales Of Nowhere podcast.  

Deployed with Heroku at http://ton-soundboard.herokuapp.com  
Check out the podcast at https://www.talesofnowhere.com  

To host the soundboard locally, simple run `run.bat` or `run.sh`.  
Audio files and button images can be found here: `ton_soundboard/static/button_files_original_volume`.  
Button audio volume is adjusted to maintain homogeneous volume across the board (pun intended).
For details, see `ton_soundboard/scripts/adjust_audio_volume.py`.  
Made with Bokeh Server.
